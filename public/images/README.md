# Images Directory

## Required Images for Hospital Website

### Minimum Required Images (9 images):

1. **hero-bg.jpg** (1920x1080px recommended)
   - Hero section background for homepage
   - Professional hospital exterior or medical facility

2. **hospital-building.jpg** (1200x800px recommended)
   - Main hospital building/facility image
   - Used on homepage and about page

3. **department-opd.jpg** (600x400px recommended)
   - Outpatient Department image

4. **department-pharmacy.jpg** (600x400px recommended)
   - Pharmacy department image

5. **department-laboratory.jpg** (600x400px recommended)
   - Laboratory department image

6. **department-maternity.jpg** (600x400px recommended)
   - Maternity department image

7. **department-emergency.jpg** (600x400px recommended)
   - Emergency department image

8. **department-surgery.jpg** (600x400px recommended)
   - Surgery department image

9. **doctor-placeholder.jpg** (400x400px recommended)
   - Default placeholder for doctor profile photos
   - Square format for consistent display

### Optional Images (for enhanced design):

10. **about-hospital.jpg** (1200x600px)
    - Additional image for about page

11. **contact-map.jpg** (800x600px)
    - Hospital location map image (if not using Google Maps)

12. **facilities-1.jpg, facilities-2.jpg, facilities-3.jpg**
    - Additional facility images for homepage or about page

## Image Guidelines:

- **Format**: JPG or PNG
- **Optimization**: Compress images for web (use tools like TinyPNG)
- **Naming**: Use lowercase with hyphens (e.g., `hospital-building.jpg`)
- **Alt Text**: All images should have descriptive alt text for accessibility
- **Responsive**: Images will be automatically responsive via CSS

## Where to Get Images:

1. **Free Stock Photos**:
   - Unsplash (unsplash.com) - Search "hospital", "medical", "doctor"
   - Pexels (pexels.com)
   - Pixabay (pixabay.com)

2. **Medical Stock Photos**:
   - Search for: "hospital building", "medical facility", "doctor", "pharmacy", "laboratory", "emergency room", "surgery room", "maternity ward"

3. **Keywords for Search**:
   - Hospital exterior
   - Modern medical facility
   - Doctor consultation
   - Pharmacy interior
   - Laboratory equipment
   - Emergency room
   - Operating room
   - Maternity ward
